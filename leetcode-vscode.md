修改插件配置，设置文件夹目录为`LeetCode/all`，这样才会自动创建go文件在此

``` json
"leetcode.filePath": {
        "default": {
            "folder": "LeetCode/all",
            "filename": "${id}.${kebab-case-name}.${ext}"
        }
    }
```
